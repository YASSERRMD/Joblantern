# IRB / Ethics Review Checklist

For Tier C (restricted) research partnerships. Tier B partnerships
fill in only the relevant rows.

- [ ] Principal investigator named.
- [ ] Institutional ethics body identified.
- [ ] IRB approval letter on file (or equivalent — e.g., REC in EU).
- [ ] Research question stated in plain language.
- [ ] No subject-level re-identification attempted.
- [ ] Data minimisation: only the fields necessary for the question are requested.
- [ ] Storage location compliant with the residency policy.
- [ ] End-of-research data-destruction plan documented.
- [ ] Conflict-of-interest declarations signed by the PI.
- [ ] Pre-registration filed (see [PRE-REGISTRATION](PRE-REGISTRATION.md)).
- [ ] Output review window agreed.
