# MCP Server — `joblantern.registry.uk`

UK-only Companies House variant of `mcp-registry`. Same tool surface
(`lookup_company`, `get_company`, `check_registration_status`) but
backed by the UK government's free Companies House API.

Auth: `COMPANIES_HOUSE_API_KEY` env var.

Attribution: data is OGL v3; display "Contains public sector
information licensed under the Open Government Licence v3.0" on any
page that surfaces a record.
