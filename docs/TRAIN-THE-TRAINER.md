# Train-the-Trainer Materials

Resources so a single skilled partner can teach others. The model is
to grow a network of regional trainers; Joblantern central staff
stays small.

## Skill ladder

1. **Operator** — runs a Joblantern instance.
2. **Senior operator** — also helps onboard new operators.
3. **Trainer** — runs the curriculum for a cohort of new partners.
4. **Coordinator** — leads a regional working group.

## Trainer kit

- The full curriculum (offline package).
- Speaker notes per module.
- Common-question handbook.
- Localised glossary for each supported language.

## Certification

After running two successful cohorts, a senior operator can submit
a short reflection to the council to be listed as a trainer.

## Cadence

- Annual trainer summit (virtual, hybrid).
- Quarterly trainer check-in per region.
