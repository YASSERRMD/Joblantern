# Marketplace Module — Legal Review Per Jurisdiction

Each marketplace platform we ingest has its own ToS and each
jurisdiction has its own consumer-protection regime. Before
activating an ingestion source we document the per-jurisdiction
posture.

## Per-platform checklist

- [ ] ToS allows automated reads of the data we use?
- [ ] If not, is there a published partner API or data-feed?
- [ ] What rate limits apply?
- [ ] Are there fields we are explicitly prohibited from storing?
- [ ] Does the platform require attribution on downstream surfaces?

## Per-jurisdiction checklist

- [ ] Consumer-protection regulator identified.
- [ ] Local definition of an "illegal commercial practice" reviewed.
- [ ] Local complaint-forwarding pathway documented (Phase 37).
- [ ] Privacy posture aligns with Phase 48 compliance.

## Audit cadence

Annual review, plus an ad-hoc review whenever a platform changes ToS.
