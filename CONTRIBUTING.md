# Contributing to Joblantern

Welcome, and thank you for helping make Joblantern better.

- Sign your commits with `git commit -s` (we use the [DCO](https://developercertificate.org/)).
- Run `make fmt lint test` before pushing.
- Open a PR from a feature or phase branch — never push directly to `main`.
- All dependencies must be Apache 2.0, BSD, MIT, or another permissive license.

By participating in this project you agree to abide by the [Code of Conduct](CODE_OF_CONDUCT.md).
