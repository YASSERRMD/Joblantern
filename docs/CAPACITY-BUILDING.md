# Capacity Building

Joblantern is more impactful when local NGOs run their own
instances. This document is the front door to the partner programme.

## Programme shape

- **Self-serve onboarding** under 2 hours — see [onboarding](../internal/capacity/onboarding/selfserve.go).
- **Curriculum** ≈ 3 hours of structured material — see [curriculum](../internal/capacity/curriculum/modules.go).
- **Mentor matching** at signup — see [mentor](../internal/capacity/mentor/match.go).
- **Regional working groups** — see [groups](../internal/capacity/groups/regional.go).
- **Office hours** monthly — see [officehours](../internal/capacity/officehours/schedule.go).
- **Train-the-trainer** — see [TRAIN-THE-TRAINER](TRAIN-THE-TRAINER.md).
- **Grant toolkit** — see [GRANT-TOOLKIT](GRANT-TOOLKIT.md).

## Quality bar

- Verdicts delivered locally must meet the same compliance posture as the hosted service.
- Localised content is reviewed by the regional working group.

## Time-to-first-verdict

The shipped target is one working day for an NGO with basic Linux
operations capacity. The deploy-in-a-box images are the main lever.
