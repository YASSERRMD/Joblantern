# Data-Sharing Agreement Templates

We publish three templates so the appropriate tier is obvious.

## Tier A — Aggregated

- Purpose: any public anonymised dashboard or PSA.
- Mechanism: link to existing public exports.
- Agreement: not required.

## Tier B — Pseudonymised

- Purpose: statistical analysis without identifiers.
- Mechanism: pseudonymised verdict snapshots delivered monthly.
- Agreement: short DSA, listed below.
- IRB required: usually no, but check local rules.

## Tier C — Restricted

- Purpose: longitudinal analysis or text-mining requiring richer fields.
- Mechanism: sandboxed replica access for a fixed window.
- Agreement: full DSA, IRB approval required, named PI.
- Output review: Joblantern may request pre-publication review for
  re-identification risk only (not for findings).

## Common clauses

- Purpose limitation.
- No attempt to re-identify subjects.
- Sub-licensing prohibited.
- Joblantern attribution in publications.
- Data destruction at the end of the research window.
