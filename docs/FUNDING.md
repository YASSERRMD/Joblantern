# Funding

Joblantern is Apache 2.0 and free to use. Operating costs (Postgres
host, optional Nominatim/Overpass tiles, optional commercial registry
API tokens) are paid by the operator.

This document is a placeholder. A future PR — opened by a human, with
real legal entities and bank details — wires up an Open Collective
and / or GitHub Sponsors page and adds a `.github/FUNDING.yml`
manifest. The repository does **not** auto-publish a funding link
until that PR lands.

The Maintainer accepts no direct payment for code contributions to
this repository.
