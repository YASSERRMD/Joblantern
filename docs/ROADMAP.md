# Roadmap

## v0.1.0 (May 2026)

End-to-end scaffold + 8 MCP servers + agent orchestrator + risk engine +
basic HTML UI. See [CHANGELOG.md](../CHANGELOG.md).

## v0.2.0

- Wire at least one LLM provider for natural-language explanation
  alongside the deterministic risk engine.
- Templ + HTMX rewrite of the HTML pages for progressive enhancement.
- Leaflet map on the result page.
- More registry providers (UK Companies House, US SEC EDGAR).
- GitHub Actions release workflow → ghcr.io multi-arch images.

## v0.3+

- A2A protocol for federating Joblantern instances across NGOs.
- Mobile PWA + browser extension.
- On-device embedding to keep semantic search private.
- Larger scam-DB corpus from sources whose terms permit redistribution.

## Out of scope (still)

- Live identity-document upload (privacy / ethics).
- LLM fine-tuning on scam corpus.
